# AI 助手桌面软件

这是一个可以让 ChatGPT 运行在您的电脑上的软件。

您可以使用 Cmd+Shift+G (Mac) 或 Ctrl+Shift+G (Win) 从任何地方快速打开它。

下载:

- [Win .exe](https://github.com/diorlitao/ai-assistant/releases/download/v0.0.9/ai-assistant-0.0.9.exe)
